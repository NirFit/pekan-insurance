# דוח ביקורת קידום (SEO) – פקאן סוכנות לביטוח

**אתר:** https://nirfit.github.io/pekan-insurance/  
**תאריך:** 12 במרץ 2025

---

## סיכום מנהלים

האתר בנוי היטב מבחינת SEO בסיסי: יש meta tags, structured data, sitemap ו-robots.txt. עם זאת, קיימות הזדמנויות משמעותיות לשיפור הדירוג במנועי חיפוש, במיוחד בחיפושים מקומיים (ביטוח עפולה, ביטוח רחובות) ובחיפושי מידע (ביטוח בריאות, ביטוח נסיעות).

---

## 1. מה כבר עובד טוב ✅

| אלמנט | סטטוס |
|-------|--------|
| **Title** | מכיל מילות מפתח רלוונטיות (פקאן, ביטוח, עפולה, רחובות, 2015) |
| **Meta description** | תיאור ברור עם CTA (ייעוץ חינם) |
| **Keywords** | מוגדרים – פקאן ביטוח, ביטוח עפולה, ביטוח רחובות |
| **Canonical URL** | מוגדר נכון |
| **Open Graph** | og:title, og:description, og:image, og:locale |
| **Schema.org** | InsuranceAgency עם פרטי קשר, כתובות, שעות פתיחה |
| **Sitemap.xml** | קיים וכולל index + privacy |
| **Robots.txt** | מאפשר סריקה ומפנה ל-sitemap |
| **RTL + lang="he"** | תמיכה בעברית |
| **Lazy loading** | תמונות עם loading="lazy" |
| **fetchpriority="high"** | תמונת Hero |
| **Alt texts** | רוב התמונות עם תיאור |
| **נגישות** | Skip link, ARIA, תקן 5568 |

## 2. הזדמנויות לשיפור – טכניקות קידום מומלצות

### 2.1 תוכן ו-Keywords (עדיפות גבוהה)

| פעולה | תיאור |
|-------|--------|
| **הרחבת מילות מפתח** | הוסף ל-keywords: "ביטוח חיים", "ביטוח סיעוד", "ביטוח אובדן כושר", "סוכנות ביטוח עפולה", "סוכנות ביטוח רחובות", "השוואת ביטוחים" |
| **דפי נחיתה לפי שירות** | שקול דפים נפרדים: `/ביטוח-בריאות`, `/ביטוח-נסיעות`, `/ביטוח-חיים` – כל דף עם תוכן ייחודי ומילות מפתח ממוקדות |
| **בלוג / מדור מידע** | מאמרים כמו "מה חשוב בביטוח נסיעות לחו״ל", "הבדלים בין ביטוחי בריאות" – מושכים חיפושי מידע ומחזקים סמכות |
| **תוכן מקומי** | הוסף טקסט ייחודי על כל סניף: "ביטוח בעפולה – המלאכה 21", "ביטוח ברחובות – פקיעין 2" |

### 2.2 Schema.org – Structured Data (עדיפות גבוהה)

| Schema חסר | השפעה |
|-------------|--------|
| **FAQPage** | שאלות נפוצות יופיעו כ-rich snippets בתוצאות החיפוש – קליקים חינם |
| **LocalBusiness** (מלא) | שיפור ב-Google Maps ו־Local Pack |
| **Review / AggregateRating** | כוכבים בתוצאות החיפוש – שיעור קליקים גבוה יותר |
| **BreadcrumbList** | ניווט ברור למנועי חיפוש |

**המלצה:** הוסף JSON-LD ל-FAQ (4 השאלות הקיימות) – קל ליישום ומשפיע מיד.

### 2.3 Open Graph & Social (עדיפות בינונית)

| חסר | תיקון |
|-----|--------|
| **og:image:width / og:image:height** | הוסף מידות לתמונה (למשל 1200x630) לשיפור תצוגה ברשתות |
| **og:image:alt** | תיאור לתמונה לנגישות |
| **Twitter Card** | `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image` – חשוב לשיתופים |
| **תמונת OG ייעודית** | במקום לוגו – תמונה מושכת (צוות, משרד, שירות) בגודל 1200x630 |

### 2.4 טכני

| נושא | המלצה |
|------|--------|
| **Preload ל-CSS/גופנים** | `rel="preload"` ל-styles.css ו־Heebo – שיפור LCP |
| **תמונות חיצוניות (Unsplash)** | שקול אירוח מקומי – שליטה מלאה, מהירות, יציבות |
| **index.html vs /** | וודא ש־index.html מפנה ל-canonical נכון (כבר מוגדר) |
| **Formspree** | החלף `YOUR_FORMSPREE_ID` ב-config.js – טופס חי = המרות |

### 2.5 Google Analytics & מדידה

| פעולה | תועלת |
|-------|--------|
| **הפעלת GA4** | הוסף `GA_ID` ב-config.js – מדידת תנועה, המרות, התנהגות |
| **Google Search Console** | רישום האתר – מעקב אחר ביצועים, שגיאות, כיסוי |
| **Google Business Profile** | פרופיל עסקי לעפולה ורחובות – חיפוש מקומי |

### 2.6 קישורים פנימיים וחיצוניים

| פעולה | תיאור |
|-------|--------|
| **קישור מ-privacy ל-index** | כבר קיים – טוב |
| **קישורים לדפי שירות** | כשנוצרים – קישור מעמוד הבית |
| **Backlinks** | קישורים מאתרים רלוונטיים (מדריכים, פורטלים מקומיים) |

## 3. תיקונים מהירים (Quick Wins)

1. **הוספת FAQ Schema** – JSON-LD עם 4 השאלות מהסעיף FAQ (זמן: ~15 דקות)
2. **הוספת Twitter Card** – 4 שורות meta (זמן: ~5 דקות)
3. **הרחבת keywords** – עדכון meta (זמן: ~2 דקות)
4. **og:image ייעודי** – תמונה 1200x630 (זמן: תלוי בעיצוב)
5. **החלפת Formspree ID** – אם עדיין לא הוחלף
