# פקאן סוכנות לביטוח - אתר פרימיום

גרסה מעודכנת ומקצועית של אתר סוכנות הביטוח פקאן.

## הפעלה

```bash
# פתיחת האתר בדפדפן
open index.html

# או עם שרת מקומי:
python3 -m http.server 8080
# ואז גלוש ל: http://localhost:8080
```

## מבנה הפרויקט

- `index.html` - דף הבית עם כל התוכן
- `styles.css` - עיצוב פרימיום עם אנימציות
- `script.js` - אינטראקטיביות (מונה, תפריט, טופס)

## תכונות

- עיצוב RTL מלא לעברית
- רספונסיבי (נייד, טאבלט, דסקטופ)
- אנימציות scroll
- מונה סטטיסטיקות אנימטיבי
- טופס צור קשר
- פונט Heebo מקצועי
